# Random Password Generator

**Автор:** Попадич Алина Антоновна
**Дата создания** Апрель 2026

## Описание

Простое GUI-приложение для генерации случайных паролей с возможностью:
- Настройки длины пароля (от 4 до 50 символов)
- Выбора типов символов: цифры, буквы, спецсимволы
- Сохранения истории сгенерированных паролей в JSON
- Копирования пароля в буфер обмена
- Очистки истории

## Требования

- Python 3.6+
- Стандартные библиотеки: `tkinter`, `random`, `string`, `json`, `os`, `datetime`
